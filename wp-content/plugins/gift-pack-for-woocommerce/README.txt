== Description ==
Let customers add gift packing/wrapping to individual products from product pages. choose your own Gift Pack design directly from the product page with message for Gift Pack. set your own price for gift pack for entire site or even set custom price for each product.


= Features =

* Allow customers add gift wrapping to individual products from product page
* Add global price for gift pack for all products
* Add different price for individual product
* Enable / Disable gift wrapping option for specific product category from setting page
* Allow customer to choose amazing Gift pack layout from product page
* Write your own Greeting message to Gift Pack
* Works with both simple & variable products
* Enable / Disable Gift Pack Note
* Enable / Disable Gift Pack Images
* Allow site owner to add unlimited layout Or gift pack images for Gift Packs
* Gift Pack details to customer orders emails 
* Easy to use & customize
* Fully Compatible with WPML
* Lightbox & Popup for Giftpack images
* Option to Enable/Disable Giftpack images

== Installation ==
1. Upload `gift-pack-for-woocommerce` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. You will find Gift Pack for WooCommerce in your WordPress admin screen.
